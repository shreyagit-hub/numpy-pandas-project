# Data Analysis Project: NumPy & Pandas Showcase

This project demonstrates practical skills with NumPy and Pandas through a complete data analysis workflow.

## Project Structure
- **data_generator.py** - Generates synthetic dataset
- **data_processor.py** - Core data manipulation and processing
- **analysis.py** - Statistical analysis and insights
- **visualization.py** - Data visualization and reporting
- **main.py** - Orchestrates the entire pipeline
- **requirements.txt** - Project dependencies

## Features
### NumPy Skills
- Array creation and manipulation
- Mathematical operations and computations
- Statistical functions (mean, std, percentiles)
- Array slicing and indexing
- Random number generation

### Pandas Skills
- DataFrame creation and inspection
- Data cleaning and handling missing values
- GroupBy operations and aggregations
- Merging and joining datasets
- Time-series operations
- Data filtering and transformation

## Usage
```
bash
pip install -r requirements.txt
python main.py
```

## Output
The project generates:
- Processed dataset with statistics
- Data quality report
- Performance metrics by category
- Summary visualizations
